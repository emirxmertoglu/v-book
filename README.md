# v-book
A web app where you can view the top 100 classical books
I use Vue.Js & Vue Router & Json-Server with this project

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
